# HotelAPi

## Descrição

O **HotelAPi** é uma aplicação web completa para controle de solicitações e operações de hotelaria.  
O backend foi desenvolvido com **C# e .NET**, utilizando Entity Framework e SQL Server para persistência de dados, com suporte a funcionalidades em tempo real via **SignalR**.  
O frontend é uma aplicação em **React** que consome a API e oferece interface interativa ao usuário, incluindo geração de relatórios em PDF.

Este projeto foi pensado com **conceitos de SaaS e práticas de arquitetura de software**, ideal para estudos e demonstração de integração entre front-end moderno e back-end robusto.

## Funcionalidades

- Cadastro e gerenciamento de solicitações  
- Regras de negócio aplicadas no backend  
- Atualização em tempo real das solicitações via SignalR  
- Geração de relatórios em PDF  
- Interface web moderna com React
